# Online Store API

A RESTful API built using ASP.NET Core for managing an online store. This API allows users to browse products, add items to their basket, and place orders.

## Features

- **Product Management:** List and display products available for purchase.
- **Shopping Basket:** Users can add items to their basket.
- **Order Processing:** Users can place orders for items in their basket.
- **Authentication:** Secure user authentication using JWT.

## Technologies Used

- **Backend:** ASP.NET Core
- **Database:** MS SQL Server
- **ORM:** Entity Framework
- **Caching:** Redis
- **Authentication:** JWT (JSON Web Tokens)
- **Logging:** Serilog

## Getting Started

### Prerequisites

- .NET SDK (version 8.0)
- MS SQL Server
- Redis Server
- Visual Studio or any other .NET-compatible IDE
